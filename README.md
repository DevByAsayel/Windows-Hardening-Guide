# Windows 11 Hardening Guide 

This project provides a beginner-friendly guide for securing **Windows 11** systems by following essential security best practices. It focuses on reducing security risks, minimizing attack surfaces, and improving overall system protection.

##  Project Overview

The purpose of this project is to help users understand how to:

* Perform a secure installation of Windows 11
* Disable unnecessary and risky services
* Keep the system updated through proper patching
* Manage user accounts and security policies
* Enable essential built-in security features

## Contents

* Secure Installation
* Service Hardening
* Patching and Updates
* User Accounts and Policies
* Additional Security Recommendations
